Expense Tracker CLI

Overview
This project is a simple command-line application that helps users track their daily expenses. It allows users to add, view, and manage expenses easily using a terminal. The main purpose is to help users understand where their money is going and manage their budget better.

---

Objective
The goal of this project is to create a lightweight system that:

* Tracks daily expenses
* Helps manage monthly budget
* Shows spending patterns

---

Features

1. Add Expense
   User can add expense with amount, category, and date

2. View Expenses
   Displays all saved expenses

3. Category Summary
   Shows total spending for each category (Food, Travel, etc.)

4. Monthly Total
   Calculates total expenses for a selected month

5. Budget Feature
   User can set a monthly budget and get alert if it exceeds

6. Delete Expense
   Remove any incorrect entry

7. Filter Expenses
   Filter expenses by category or date

8. Top Spending Category
   Shows where most money is spent

9. Export to CSV
   Exports all data to a CSV file

---

Technology Used

* Python
* JSON for data storage
* CLI (Command Line Interface)


---

How It Works

* User selects an option from menu
* Program takes input
* Data is stored in JSON files
* Output is shown in terminal

---

Example

Add Expense → Enter amount, category, date
View Expenses → Shows all saved data
Set Budget → Alerts if exceeded
